AI interfaces are fundamentally different from deterministic software. A button always does the same thing. An LLM doesn't.

This chapter shows you how to design AI features that are useful rather than frustrating — and that users trust, even when results aren't always perfect.

## What you'll learn

- How AI-native UX patterns work (and why a chatbot is often the wrong choice)
- How to build trust when results aren't deterministic
- Which interaction patterns between human and AI work
- How to design generative features that manage expectations
