- Wie Multi-Agent Systeme funktionieren und wann sie Sinn machen
- Was Tool Use und MCP bedeuten — und warum sie die Faehigkeiten von AI erweitern
- Welche Autonomie-Level es gibt (Vorschlag → Ausfuehrung → Eigenstaendig)
- Wann ein Mensch im Loop bleiben muss
