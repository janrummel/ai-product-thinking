# Design: AI Product Thinking

> Approved: 2026-03-10
> Approach: "The Publication" — Dieter Rams / Jony Ive inspired

## Identity

| Aspect | Decision |

| Name | AI Product Thinking |

| Tagline | "Die Prinzipien hinter AI-Produkten verstehen." |

## Visual Language

### Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Background | Warm Ivory | `#F5F3EE` | Page background (Light) |
| Text | Anthracite | `#2C2C2C` | Body text, headings |
| Secondary | Warm Gray | `#8A8680` | Meta text, inactive sidebar, dividers |
| Accent | Terracotta | `#B85C38` | Links, active chapter, CTAs, case study markers |
| Code BG | Sand Gray | `#EDEBE6` | Code blocks, inline code |
| Dark BG | Warm Gray | `#1A1A18` | Dark mode background |
| Dark Text | Warm White | `#E8E6E1` | Dark mode text |

### Design Principles

- Light mode default
- System fonts (no custom fonts)
- Max content width: 42rem (readable measure)
- Generous whitespace, generous line-height (1.75)
- No emoji in headings
- No colored boxes or gradients
- No badges in sidebar
- No icons — numbers and words suffice
- Diagrams only when a sentence doesn't suffice
- Every element must serve a function

## Structure

### Naming

- Chapters (not Levels)
- Lessons (not Challenges)
- Synthesis (not Boss Fight)
- Purely numerical: `01 — Foundations`, `02 — Strategy`

Per chapter: 1 introduction + 4 lessons + 1 synthesis = 6 files

## Lesson Pattern: CONTEXT → CONCEPT → FRAMEWORK → SCENARIO → DECIDE → REFLECT

Each lesson flows through 6 steps. They are NOT visually separated blocks — they flow naturally, separated only by H2 headings.

1. **Context** — Real situation as entry point
2. **Concept** — Theory behind it, with sources
3. **Framework** — A decision tool to take away
4. **Scenario** — Apply the framework to the entry situation
5. **Decide** — Own decision via `<details>` element
6. **Reflect** — Key insight, connection to next topic

## Quality Plan

### Source Hierarchy

| Rank | Source |
|------|--------|
| 1 | Peer-reviewed research, official documentation |
| 2 | Established frameworks (Reforge, Lenny, SVPG) |
| 3 | Practitioner content (blog posts from experienced AI PMs) |
| 4 | Course content (Maven, Product School — structure, not copy) |

Rule: No content without Rank 1-2 source. Rank 3-4 supplementary only.

## Tech Stack

|-----------|--------|--------|

## Contrast to level-up-ai

| Aspect | level-up-ai | AI Product Thinking |
|--------|-------------|---------------------|
| Aesthetic | Gaming, colorful, emoji-heavy | Publication, minimal, Rams/Ive |
| Color | Orange/yellow gradients | Ivory + terracotta accent |
| Structure | Levels, Challenges, Boss Fights | Chapters, Lessons, Synthesis |
| Mode | Dark default | Light default |
| Content | Code exercises | Decision scenarios |
| Pattern | THINK→OVERVIEW→WHY→WALKTHROUGH→TRY→COMBINE | CONTEXT→CONCEPT→FRAMEWORK→SCENARIO→DECIDE→REFLECT |
